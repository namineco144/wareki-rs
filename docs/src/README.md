# wareki-rs とは

`wareki-rs` は、西暦（グレゴリオ暦）と和暦（Japanese Era）を相互変換するための高速かつ安全なライブラリです。

コアロジックは **Rust** で実装されており、日付処理には広く使われている `chrono` クレートを採用しています。これにより、以下のような恩恵があります。

- **厳密な日付評価**: 架空の閏日（例: 閏年でない年の2月29日）を入力した場合は正しく弾かれます
- **正確な元号境界の処理**: 明治、大正、昭和、平成、令和の改元のタイミングを厳密に計算します
- **安全なメモリ管理と高速な実行**: 各言語のネイティブ拡張として動作します

## 特徴
`from_wareki` メソッドなどによる和暦から西暦への変換では、以下のように複数の入力フォーマットを柔軟に受け入れます。

- フルネーム: `令和`, `平成`
- 漢字一文字: `令`, `平`
- ローマ字（大文字・小文字両対応）: `R`, `r`, `H`, `h`
